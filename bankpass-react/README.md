# BankPass+ React Application

Premium Fintech Verification Flow with Wallet Integration and AI/OCR

## Setup

```bash
npm install
npm run dev
```

## Features

### Updated Flow:
1. **Wallet Connection** - Connect MetaMask, WalletConnect, or built-in wallet
2. **Service Selection** - Choose "Update Phone Number" or other services
3. **Passbook Upload** - Upload passbook image
4. **AI/OCR Extraction** - Tesseract.js extracts Account Number and Name
5. **Review Data** - Verify extracted information
6. **ID Verification** - Upload ID proof
7. **AI Analysis** - Automated verification
8. **Status Tracking** - Real-time approval status
9. **Success** - Download report

### Key Features:
- Web3 wallet integration (MetaMask, WalletConnect)
- Local OCR with Tesseract.js
- Real-time extraction progress
- Glassmorphism UI design
- Fully responsive

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Tesseract.js (OCR)
- Web3 Wallet Integration
- Font Awesome Icons
